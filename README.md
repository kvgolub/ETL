# ETL (Python, SQL, PostgreSQL)

Переносимые данные расположены в файлах в папке data.

1. Устанавить зависимости из req.txt
2. Отредактировать настройки подключения к PostgreSQL в main.py
3. запустить скрикты для создания БД и новых таблиц: create_database.sql, create_static_data.sql, create_tables.sql (Справка во скриптам в файле sql_script/Описание доп.скритптов.txt)
4. Запуск ETL через скрипт start.sh или main.py
